# 📊 Cert Study Hub — PL-300 & DP-700

Web app de estudio para las certificaciones Microsoft **PL-300** (Power BI Data Analyst) y **DP-700** (Fabric Data Engineer Associate).

## 🚀 Cómo usar

### Opción A — Directamente en el navegador
Descargá `index.html` y abrilo con doble clic. Funciona 100% offline.

### Opción B — GitHub Pages (recomendado)
1. Creá un repo en GitHub (ej: `cert-study-hub`)
2. Subí `index.html` y `README.md`
3. Andá a **Settings → Pages → Source → main branch → / (root)**
4. Tu hub queda disponible en `https://TU-USUARIO.github.io/cert-study-hub/`

- 📅 Plan semanal (viernes + sábados)
- 📖 Teoría clave con 10 conceptos fundamentales

- 🔗 Recursos gratuitos curados

## 🔧 Features
- Responsive (móvil + desktop)
- Barra de navegación inferior en móvil
- Checks persistentes con localStorage
- Contador de días hasta el examen (editable)
- Funciona 100% sin internet (excepto links externos)

## 📝 Notas
- Requiere un navegador moderno (Chrome, Firefox, Edge, Safari)
- El localStorage guarda los checks en el mismo navegador/dispositivo
